🏡 #Homestay ReviewAI

📌 Overview
Homestay ReviewAI is a full-stack web application designed to collect, manage, and display guest reviews for homestay properties. The platform enables users to submit reviews and view all feedback in real time. It uses a MERN-style architecture with React on the frontend, Node.js + Express on the backend, and MongoDB for data storage.
The project focuses on building a scalable review system with REST API integration and dynamic frontend-backend communication.

✨ Features
📝 Review Management
Users can submit reviews through a simple UI
Reviews are stored in MongoDB database
All reviews are fetched and displayed dynamically
🔗 Full Stack Integration
React frontend connected with Express backend
REST API communication using Axios
Real-time data updates between frontend and backend
🗄️ Database Integration
MongoDB Atlas used for cloud database
Mongoose schema validation for structured data
⚙️ Backend APIs
GET API to fetch all reviews
POST API to add new reviews
Error handling for invalid or missing data
💻 Frontend Interface
Dashboard to display all reviews
Input section to submit new reviews
Clean and responsive UI using React

🚀 How to Run the Project Locally
1. Clone Repository
git clone https://github.com/Ketna-Gupta/homestay-review-ai.git
2. Backend Setup
cd homestay-review-ai/backend
npm install
Create .env file:
PORT=5000
MONGO_URI=your_mongodb_connection_string
Start backend:
npm run dev
3.Frontend Setup
cd homestay-review-ai/frontend
npm install
npm run dev
